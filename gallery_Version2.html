````markdown name=README.md ```markdown # wearethesapace — Art Grid Designer This is a small client-side site for an art installation. Visitors can choose a grid (3x3, 6x6, 9x9), paint cells using a palette or color wheel, and submit their design. Submissions are stored in the browser's localStorage and displayed on the gallery page. Files: - index.html — designer - gallery.html — combined gallery of submissions - style.css — styles - app.js — client-side logic Open index.html locally to test, or host with GitHub Pages.
